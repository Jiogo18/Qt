Control d'un ordinateur à distance

L'ordinateur controlé utilise serveurTrack.exe
L'ordinateur de control utilise clientTrack.exe

Le control peut etre arreté en appuyant sur Key_Alt sur la fenètre
Le serveurTrack peut relancer le control en appuyant sur Key_Control

Client DLL:
platforms/qwindows.dll
libgcc_s_dw2-1.dll
libstdc++-6.dll
libwinpthread-1.dll
Qt5Core.dll
Qt5Gui.dll
Qt5Network.dll
Qt5Widgets.dll
